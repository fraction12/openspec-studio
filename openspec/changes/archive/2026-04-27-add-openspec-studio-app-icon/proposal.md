# Add OpenSpec Studio app icon

## Summary
Create a dedicated OpenSpec Studio icon and in-app logo mark that feels native to the OpenSpec website and product language.

## Motivation
OpenSpec Studio still uses generic/default icon assets and a text-only "OS" rail mark. The desktop app needs a recognizable identity that fits the restrained OpenSpec visual system: neutral surfaces, mono-code detail, repository/spec artifacts, and quiet semantic accents.

## Scope
- Add a source SVG app icon/logo mark based on the OpenSpec website's neutral, document-first style.
- Generate the Tauri icon assets used by the packaged desktop app.
- Use the same mark in the left rail brand area so the in-app identity matches the bundle icon.

## Non-goals
- Rebrand OpenSpec itself.
- Add decorative illustration to product surfaces.
- Change app navigation, layout, or product behavior.
